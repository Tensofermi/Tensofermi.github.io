索引、reshape、广播与 inplace 操作¶

掌握 tensor 操作的重点不是记住所有 API，而是理解 shape 如何变化。

这一页覆盖最常见的四类操作：

  • 索引和切片。
  • reshape / view。
  • transpose / permute。
  • broadcasting。
  • inplace 操作的风险。

索引和切片¶

import torch

x = torch.arange(12).reshape(3, 4)
print(x)
print(x[0])
print(x[:, 1])
print(x[1:, 2:])

x[0] 取第 0 行，shape 从 \([3,4]\) 变成 \([4]\)。
x[:,1] 取所有行的第 1 列，shape 也是 \([3]\)。

如果想保留维度，可以写：

col = x[:, 1:2]
print(col.shape)

得到：

torch.Size([3, 1])

reshape 和 view¶

reshape 改变 tensor 形状，但元素总数必须不变。

x = torch.arange(24)
y = x.reshape(2, 3, 4)
print(y.shape)

元素数：

\[ 24=2\times3\times4. \]

可以用 -1 自动推断：

z = y.reshape(6, -1)
print(z.shape)

得到 \([6,4]\)。

view 和 reshape 类似，但要求内存连续。一般初学时优先用 reshape。

transpose 和 permute¶

矩阵转置：

x = torch.randn(2, 3)
y = x.transpose(0, 1)
print(y.shape)

高维换轴用 permute：

x = torch.randn(2, 3, 4)
y = x.permute(0, 2, 1)
print(y.shape)

如果图像数据从 \([B,H,W,C]\) 变成 PyTorch 常用的 \([B,C,H,W]\)：

images = torch.randn(8, 32, 32, 3)
images = images.permute(0, 3, 1, 2)
print(images.shape)

unsqueeze 和 squeeze¶

unsqueeze 增加长度为 1 的维度：

x = torch.randn(5)
y = x.unsqueeze(0)
z = x.unsqueeze(1)

print(y.shape)
print(z.shape)

输出：

torch.Size([1, 5])
torch.Size([5, 1])

squeeze 删除长度为 1 的维度：

x = torch.randn(1, 5, 1)
print(x.squeeze().shape)

Broadcasting¶

广播允许不同 shape 的 tensor 自动对齐。

x = torch.randn(2, 3)
b = torch.randn(3)
y = x + b
print(y.shape)

这里 \(b\) 的 shape 是 \([3]\)，会广播到 \([2,3]\)。

广播规则从右往左对齐。两个维度兼容当且仅当：

  • 相等。
  • 其中一个为 1。
  • 其中一个不存在。

例子：

[B, N, D]
[      D]  -> 可以广播
[   N, 1]  -> 可以广播
[B, 1, D]  -> 可以广播
[B, D]     -> 通常不符合预期

Transformer 中位置 embedding 常用：

\[ X_{\rm tok}\in\mathbb{R}^{B\times N\times D}, \qquad X_{\rm pos}\in\mathbb{R}^{1\times N\times D}. \]

相加时 \(X_{\rm pos}\) 沿 batch 维广播。

Inplace 操作¶

带下划线的操作通常是 inplace：

x = torch.ones(3)
x.add_(2)
print(x)

inplace 会直接修改原 tensor。

这可能节省内存，但在 autograd 中要小心。下面的代码可能破坏计算图需要的中间值：

x = torch.randn(3, requires_grad=True)
y = x * x

# 不推荐随意 inplace 修改参与梯度计算的张量
# x.add_(1.0)

实践建议：

  • 初学时少用 inplace。
  • 需要节省显存时再明确使用。
  • 如果遇到 autograd 报错，优先检查 inplace 操作。

最小示例：整理序列 batch¶

import torch

B, N, D = 2, 3, 4
x = torch.randn(B, N, D)

# 合并 batch 和 sequence，常用于逐 token MLP
y = x.reshape(B * N, D)
print(y.shape)

# 处理后再还原
z = y.reshape(B, N, D)
print(z.shape)

shape 操作不改变数学含义，前提是你清楚每个维度代表什么。
