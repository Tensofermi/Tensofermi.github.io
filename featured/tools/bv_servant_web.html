Support Notes

Available here: BV lookup, per-part selection, filename override, save directory, progress display, cancel, multi-thread download, optional MP3 conversion.
Backend model: the browser is only the UI. All Bilibili API calls and the actual download work happen inside the local Python service.
What you need to run: start python featured/tools/bv_servant_backend.py, then open this page and keep the backend URL pointed to that local service.
