Static-first BV toolbox for pages that cannot run a backend.

Use this when you want something deployable on GitHub Pages with zero server setup. It will normalize BV input, generate the canonical Bilibili page link, build local commands for tools like yt-dlp or your Python backend, and keep a small local history for repeated use.
